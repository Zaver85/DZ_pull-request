# Подсказка по GIT

Создание репозитория:
```sh
git init
```
Добавление изменений в коммит:
```sh
git add
```
Добавление коммита:
```sh
git commit -m "Message"
```
Посмотреть все коммиты:
```sh
git log
```
Посмотреть все коммиты в  компактном виде:
```sh
git log --oneline
```
Переключение между коммитами:
```sh
git checkout <branch_name>
```
